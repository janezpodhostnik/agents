---
name: git-staged-review
description: Instructions for the agent to review the currently staged changes as a code reviewer.
compatibility: Requires git and a repository with staged changes
---

Go trough the folowing checklist:

- [ ] Read all staged changes
- [ ] Review the code/documentation chages an look for:
  - [ ] any potential issues with the code/documentation
  - [ ] any changes that amight be missing
  - [ ] any inconsistencies according to the current code/structure/architexture
- [ ] Present the findings to the user. And offer to update AGENTS.md if it needs updating
